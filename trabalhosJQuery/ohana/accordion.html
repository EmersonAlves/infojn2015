PHP

PHP (Um acrônimo recursivo para "PHP: Hypertext preprocessor", originalmente Personal Home Page) é uma linguagem interpretada livre e utilizada para gerar conteúdo dinâmico na World Wide Web.

CMS

Content Management Systems (CMS) mas igualmente designada como Convernational Monitor System [nota 1]; um sistema gestor de Websites, e intranet's que integra ferramentas necessárias para criar, gerir (inserir e editar) conteúdos em tempo real sem a necessidade de progamação,cujo objetivo é estruturar e facilitar a criação, administração, distribuição, publicação e disponibilidade da informação.

HTML

HTML (Acrônimo para a expressão inglesa HyperText Markup Language, que significa Linguagem de Marcação de Hipertexto) é uma linguagem de marcação utilizada para produzir páginas na Web.
